    DB SETUP -------------->   DB formation
    MODEL    -------------->   COLLECTION

MODELS & COLLECTIONS:
    variety of data ko banane ke lie hum banate hain models code mein, jo code hum likhte hain wo MODEL hota hai aur us code se jo result formation ho db mein usko hum bolte hain COLLECTIONS.


SCHEMA:
    code mein hum SCHEMA banate hain aur wo result deta hai as documents in databases., ek app ka poora data kehlata hai database. jismein variety of data hota hai jo kehlata hai collections and us collections ke bhi different varieties hote hain jo hota hai schema
    aik single user ka schema bhi aik seperate document banata hai.
